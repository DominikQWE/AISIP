# Методи й засоби інтелектуальної обробки сигналів

Репозиторій містить лабораторні роботи з дисципліни  
**«Методи й засоби інтелектуальної обробки сигналів»**.

Роботи виконуються мовою **Python** у **Jupyter Notebook** (`.ipynb`) з використанням бібліотек:

- OpenCV;
- NumPy;
- Matplotlib;
- SciPy;
- Scikit-image.

## Вимоги до оформлення

- **Код і експерименти:** основний файл лабораторної — `Lab_XX/Lab_XX.ipynb` (покрокові комірки, українські коментарі, вивід зображень, збереження у `results/`).
- **Звіт:** окремо у форматі **Markdown** — файл `report.md` у тій самій папці `Lab_XX/`.

У репозиторії не використовуються та не зберігаються файли таких форматів:

- `.pdf`
- `.doc`
- `.docx`
- `.zip`
- `.rar`

## Вимоги до середовища

- **ОС:** Windows 10/11.
- **Python:** 3.10 або новіше (рекомендовано актуальну стабільну версію з [python.org](https://www.python.org/downloads/windows/)).
- **Jupyter:** залежність `jupyter` з `requirements.txt` (запуск через JupyterLab / класичний Notebook або вбудований переглядач у Cursor/VS Code).

## Інструкція запуску на Windows

1. Клонуйте репозиторій або відкрийте папку `AISIP` у Cursor.
2. У PowerShell у корені репозиторію:

   ```powershell
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   python -m pip install --upgrade pip
   pip install -r requirements.txt
   ```

3. Переконайтеся, що у корені є `satir.jpg`.
4. Запуск Jupyter (приклад — JupyterLab):

   ```powershell
   jupyter lab
   ```

   Відкрийте потрібний файл, наприклад `Lab_01/Lab_01.ipynb`, і виконуйте комірки послідовно (**Run All** або по одній). Результати зображень для звіту зберігаються у `Lab_XX/results/`.

5. Альтернатива: відкрити `.ipynb` у Cursor/VS Code і виконувати комірки в інтегрованому ноутбуку (потрібне розширення Jupyter, якщо ще не встановлене).

## Примітки

- Звіти оформлюються лише у **Markdown** (`report.md`), з ілюстраціями з папки `results`.
- Файли **PDF / DOC / DOCX / ZIP / RAR** у репозиторій не додаються; відповідні розширення ігноруються через `.gitignore`.
- Окремі `.py`-скрипти не є обов’язковими; додавайте їх лише за потреби (наприклад, допоміжні модулі), основна робота — у `.ipynb`.
- Локальні артефакти можна тримати у `_excluded_files` (папка в `.gitignore`).
