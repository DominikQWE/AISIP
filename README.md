# Методи й засоби інтелектуальної обробки сигналів

Репозиторій містить лабораторні роботи з дисципліни  
**«Методи й засоби інтелектуальної обробки сигналів»**.

У роботах розглядаються базові методи цифрової обробки зображень:
аналіз гістограм, контрастування, фільтрація, відновлення, просторові
перетворення, стиснення та блокова JPEG-подібна обробка.

## Використані засоби

Лабораторні роботи виконані мовою **Python** у середовищі **Jupyter Notebook**.

Основні бібліотеки:

- OpenCV;
- NumPy;
- Matplotlib;
- SciPy;
- Scikit-image.

## Вимоги до оформлення

Для кожної лабораторної роботи створено окрему папку `Lab_XX`.

У кожній папці містяться:

- `Lab_XX.ipynb` — ноутбук з програмною реалізацією;
- `report.md` — звіт у форматі Markdown;
- `results/` — збережені результати виконання роботи.

У репозиторії не використовуються файли таких форматів:

- `.pdf`
- `.doc`
- `.docx`
- `.zip`
- `.rar`
